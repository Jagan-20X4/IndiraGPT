# Assets Folder

## Logo File

Please place the official INDIRA IVF logo file here with the name:
- `indira-logo.png` (preferred)
- OR `indira-logo.jpg`
- OR `indira-logo.svg`

The logo should be:
- High resolution (at least 400px width recommended)
- PNG format with transparent background (preferred)
- Or JPG/SVG format

The application will automatically use this logo in:
- Header (main navigation)
- Login page
- All branded areas
